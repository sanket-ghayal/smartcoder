Learn CSS

CSS (Cascading Style Sheets) is used to design beautiful and responsive websites.

Start Learning

What is CSS?

CSS stands for Cascading Style Sheets. It controls colors, fonts, spacing, layouts and animations of web pages.

  • ✔ Beautiful Design
  • ✔ Responsive Layout
  • ✔ Animations
  • ✔ Fast Loading
  • ✔ Easy Styling

Features of CSS

Colors

Add attractive colors to webpages.

Fonts

Customize text appearance.

Layouts

Create responsive page layouts.

Animation

Add transition and animation effects.

Flexbox

Create flexible responsive layouts.

Grid

Build modern web layouts.

CSS Syntax

h1{

color:red;

font-size:40px;

}

CSS Selectors

Element

h1{}

Class

.box{}

ID

#header{}

Universal

*{}

CSS Colors

Color Name

h1{
color:red;
}

HEX Color

h1{
color:#0d6efd;
}

RGB Color

h1{
color:rgb(255,0,0);
}

HSL Color

h1{
color:hsl(220,100%,50%);
}

Background

CSS provides different background properties.

body{
background:#f4f7fb;
}

.box{
background:linear-gradient(
135deg,
#0d6efd,
#00c6ff);
}

Borders 

.box{

border:2px solid blue;

border-radius:10px;

}

Borders are used to create outlines around elements.

Margin 

.box{

margin:20px;

}

Margin creates space outside the element.

Padding 

.box{

padding:20px;

}

Padding creates space inside the element.

Fonts

p{

font-size:18px;

font-family:Poppins;

font-weight:600;

}

Text Properties

Align

text-align:center;

Transform

text-transform:uppercase;

Decoration

text-decoration:none;

Spacing

letter-spacing:2px;

CSS Flexbox

.container{

display:flex;

justify-content:center;

align-items:center;

}

Flexbox is used to create responsive one-dimensional layouts.

CSS Grid

.container{

display:grid;

grid-template-columns:
repeat(3,1fr);

gap:20px;

}

CSS Grid creates powerful two-dimensional layouts.

CSS Position 

.box{

position:relative;

top:20px;

left:30px;

}

Position values: Static, Relative, Absolute, Fixed and Sticky.

Display Property

Block

display:block;

Inline

display:inline;

Inline Block

display:inline-block;

None

display:none;

Transform

Scale

transform:scale(1.1);

Rotate

transform:rotate(45deg);

Translate

transform:translateX(50px);

Skew

transform:skew(20deg);

Transition

button{

transition:.4s;

}

button:hover{

background:red;

}

Animation 

@keyframes move{

from{

left:0;

}

to{

left:200px;

}

}

Animations make websites interactive and attractive.

Media Query 

@media(max-width:768px){

h1{

font-size:30px;

}

}

Media queries create responsive websites for mobile, tablet and desktop devices.

Best Practices

  • ✔ Use external CSS.
  • ✔ Keep code organized.
  • ✔ Use Flexbox & Grid.
  • ✔ Write responsive layouts.
  • ✔ Minimize duplicate code.
  • ✔ Use meaningful class names.
