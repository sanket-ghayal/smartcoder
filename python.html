What is Python?

Python is a high-level, interpreted programming language created by Guido van Rossum.

  • ✔ Free & Open Source
  • ✔ Cross Platform
  • ✔ Object Oriented
  • ✔ Large Community Support
  • ✔ Powerful Libraries

Operators

b=5

print(a+b)
print(a-b)
print(a*b)

Python supports Arithmetic, Assignment, Comparison and Logical Operators.

If Else 

age=18

if age>=18:

    print("Adult")

else:

    print("Minor")

Conditional statements help make decisions.

Loops

For Loop

for i in range(5):

    print(i)

While Loop

i=1

while i<=5:

    print(i)

    i+=1

Nested Loop

for i in range(2):

    for j in range(2):

        print(i,j)

Functions 

def add(a,b):

    return a+b

print(add(5,10))

Functions allow code reuse and improve readability.

Lists

"Mango",

Tuple

Tuples are ordered and immutable.

Dictionary

"age":20

Dictionaries store data in key-value pairs.

Set 

numbers={1,2,3,4}

Sets store unique values only.

Classes & Objects

class Student:

    def __init__(self,name):

        self.name=name

s=Student("Rahul")

print(s.name)

Python supports Object-Oriented Programming (OOP).

File Handling

print(file.read())

file.close()

Python can create, read, write and delete files.

Exception Handling 

try:

    print(10/0)

except:

    print("Error")

finally:

    print("Done")

Exceptions prevent programs from crashing unexpectedly.

Modules

Modules provide reusable functions and classes.
