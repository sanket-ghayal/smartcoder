What is JavaScript?

JavaScript is a lightweight scripting language used to create interactive web applications.

  • ✔ Fast
  • ✔ Lightweight
  • ✔ Dynamic
  • ✔ Cross Platform
  • ✔ Easy to Learn

Features of JavaScript

Dynamic

Create interactive websites.

Client Side

Runs inside the browser.

Lightweight

Easy and fast scripting language.

Cross Platform

Works on every operating system.

Object Oriented

Supports OOP concepts.

Event Driven

Responds to user actions.
